# Weverse O2O PM Portfolio

위버스 지원을 위한 맞춤형 포트폴리오입니다.

## 🎯 프로젝트 개요

**Fan-Centric Product Manager**를 위한 포트폴리오로, 플라피나 가이드라인에 따라 제작되었습니다.

### 핵심 전략
- ✅ 위버스 브랜드 컬러 적용 (Green #00D664, Black, White)
- ✅ 원페이저 전략 (표지에서 핵심 성과 즉시 전달)
- ✅ 12페이지 구성 (영화 예고편처럼 임팩트 있는 프로젝트만)
- ✅ 비즈니스 언어 사용 (숫자와 성과 중심)

## 📊 주요 성과

- **510K** 방문객 운영 (16일, 무사고)
- **ZERO** 재고 폐기율 (3차 연속 완판)
- **D-13** 긴급 행사 기획 ("역대급" 평가)

## 📁 파일 구조

```
.
├── index.html              # 기존 포트폴리오
├── weverse_portfolio.html  # 위버스 맞춤 포트폴리오 ⭐
└── README.md
```

## 🎨 포트폴리오 구성

### Page 1: Cover (One-Pager)
- 헤드라인: "팬덤의 마음을 읽고, Tech로 문제를 해결하는 O2O PM"
- 핵심 지표 3가지 강조
- 전체 경력 요약

### Page 2-3: 모랑공방 (Personal Work)
- **주제**: 덕질을 비즈니스로 증명
- **성과**: 3차 연속 완판, 재고 0원
- **인사이트**: 팬심 타이밍 분석 → 100% 선주문 시스템

### Page 4-5: 수원문화재단 (Collaborative Work)
- **주제**: 51만 명 트래픽을 Tech로 방어
- **성과**: 16일 무사고 운영, ChatGPT 활용 위기 대응
- **인사이트**: AI 활용 기술 문제 해결

### Page 6-7: 꼬질랜드 (Problem Solving)
- **주제**: 주최자 잠적 D-13, 위기를 놀이로 전환
- **성과**: "역대급 행사" 평가
- **인사이트**: 팬덤 코드 활용한 커뮤니케이션

### Page 8: Additional Experience
- 행정 정산 경험 (e나라도움)
- 글로벌 CS 경험
- 현장 안전 관리

### Page 9-11: Weverse Proposal
- **제안**: 위버스 아카이브 시스템
- **컨셉**: 디지털 덕질 여권
- **효과**: O2O Lock-in, 바이럴 마케팅

### Page 12: Conclusion
- 비전 제시
- 연락처 정보
